# flowable-keycloak-example

Example of Flowable UI probjects with Keycloak authentication/authorization

At this moment only `flowable-ui-modeler` is adapted to work with Keycloak in this example.

# Run it

```
mvn install
cd flowable-modeler
./start-modeler.sh
```
